Set of objects to get Google Geocode Api Service response with the coordinates of a requested address (http://maps.google.com/maps/api/geocode/json)


Retrofit sample request:

  geoApi.getCoordinates(event.getAddress().getNumber() + " " + 

      event.getAddress().getStreet() + ", " +

      event.getAddress().getCity() + ", "+

      event.getAddress().getState(), false, geocodeCallback);
